# README
## 关于memo
> memo-ui是在vue3基础上构建的一个ui库。技术栈：vue3、vite、ts、scss、tailwindcss。提供丰富的组件以便于更快更高效构建你的网站。
## 注意说明⚠️
> memo-ui的样式是基于tailwindcss构建，是在继承tailwindcss基础组件、工具类、base标签之上进行了一些样式组件扩展。所以在使用memo-ui构建网站的时候你可以使用他们。
